# wox-react-component [![image](https://img.shields.io/npm/v/wox-react-component.svg)](https://www.npmjs.com/package/wox-react-component)

## API

| 属性 | 说明 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| show | 是否显示 | Boolean | true |

## Usage

```
import WoxReactComponent from 'wox-react-component';

render(
  <WoxReactComponent />,
  rootEle
);
```
